**О чем проект**
Проект о путешествиях по стране. 
Одностраничный лэндинг из восьми блоков.
В данном проекте разработан адаптивный дизайн страницы для ширины в 320, 768, 1024 и 1280 пикселей.

**Технологии**
1.Figma;
2. Grid Layout;
3. Mediaqueries;
4.Git.
